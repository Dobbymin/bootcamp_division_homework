# 해달 0주차 과제 가이드

문제를 풀었다면 채점을 해볼 차례입니다.

## 1. C의 경우

1. 상단 메뉴바에서 `Terminal` - `New Terminal`을 클릭합니다.

    ![new terminal](../assets/5-1.png)

1. `git bash`에 아래 명령어를 입력합니다.

    ```bash
    ./grade.sh c
    ```

## 2. Python의 경우

1. 상단 메뉴바에서 `Terminal` - `New Terminal`을 클릭합니다.

    ![new terminal](../assets/5-1.png)

1. `git bash`에 아래 명령어를 입력합니다.

    ```bash
    ./grade.sh p
    ```
